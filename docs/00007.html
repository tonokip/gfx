Running the Demonstration

 

Upon boot-up, the application displays the animation Splash Screen, which shows an animated splash bar while blending in the MPLAB Harmony and Microchip PIC32 logos, as shown in the following figure. 

 

When Splash Screen animation is successful, the demonstration will display a similar menu to that shown in the following figure. 

 

 

 

When power-on is successful, the application Home Screen will be displayed 

 

 

 

The demonstration supports three configuration file input options: Program Flash, SDCard, and Host PC. ProgramFlash is the default option and requires no manual source file interaction. 

 

Program Flash

 

This is the default option. The config content is embedded in program flash of the firmware. This option requires no external file preparation. 

 

To run this option.: 

 

  1. On the Main Menu Press the Load Button.The application will transition to the State Configuration screen
  2. On the State Configuration Screen, select Program radio button. This will enable the reading of Program Flash.
  3. Press the Stage button. This step will stage and parse the configuration information located in Program Flash.
  4. Observe the Target board UI .
  5. When complete, the Stage button will be removed, and the Next screen button image will be displayed.
  6. Go to Step "All Options" below.

 

SD Card

This option requires a SDCARD. The SDCARD should be loaded with maxtouch.xcfg or maxtouch.raw file in the root folder. Only 1 file can be store on the SDCARD for this release. An example maxtouch.xcfg file can be found at the following location: <installdir>/apps/gfx_apps/aria_mxt_configure /firmware/src/config_files 

 

To run this option. 

 

  1. On the Main Menu Press the Load Button. The application will transition to the State Configuration screen.
  2. On the State Configuration Screen, select SDCARD radio button. This will enable the reading from SDCARD.
  3. Press the Stage button. The step will stage and parse the configuration information located in the maxtouch.xcfg or maxtouch.raw file.
  4. Observe the Target board UI . When complete, the Stage button will be removed, and the Next screen button image will be displayed.
  5. Go to Step "All Options" below.

 

Host PC

This option requires a HOST PC connected to the Target board. The configuration file will be streamed from the Host PC to the Target board. For the purpose of this option, the user should establish a host PC connected to the Target board via a USB cable. A USB cable needs to be connected to the micro-B USB connector on the bottom of the starter kit in use. When the demonstration runs, it will create a USB CDC device on the USB bus. The demonstration can be executed once you have connected to this device through RealTerm terminal program. See the following download link: RealTerm 

 

https://sourceforge.net/projects/realterm/ 

 

Please refer to the connection diagram shown below. 

 

To run this option.:

  1. On the Main Menu Press the Load Button. The application will transition to the State Configuration screen.
  2. On the State Configuration Screen, select PC radio button. This will enable the reading from Host PC.
  3. Press the Stage button. The step will create the USB CDC device on the USB bus and wait for a file transfer operation from the Host PC. Once initiated, the app will stage and parse the configuration. On the PC, Invoke RealTerm application .
  4. Using the Port Tab, connect to the new Serial comm port.
  5. Using the Send Tab, select the File to Port.
  6. Set Delays to 1 and 5
  7. Set Repeats 0 and 0
  8. Select the Send File button
  9. When complete, the Progress bar on the RealTerm interface will display “Done”
  10. Observe the Target board UI , When complete, the Stage button will be removed, and the Next screen button image will be displayed.
  11. Go to Step "All Options" below.

 

All Options

 

To complete the demonstration for all options.

  1. Use the Next button to transition to the Load Configuration Screen
  2. Select Load on the Load Configuration screen.
  3. Use the Next button to transition to the Key Pad Touch Accuracy test screen.
  4. Press key pad buttons
  5. Use Next to transition to the Multi Touch Test Screen
  6. Drag finger across display
  7. Use Next Button to transition to the Store Configuration (Flash) Screen
  8. Press Save
